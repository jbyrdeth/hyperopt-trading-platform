    üöÄ Quick Start¶

    Base URL¶

    http://localhost:8000/api/v1

    Authentication¶

    All requests require an API key in the header: 

    -H "X-API-Key: your_api_key_here"

    Proven Working Example¶

    This actual API call generated 45.2% returns with 1.85 Sharpe ratio:

    curl -X POST "http://localhost:8000/api/v1/optimize/single" \
  -H "X-API-Key: dev_key_123" \
  -H "Content-Type: application/json" \
  -d '{
    "strategy_name": "MovingAverageCrossover",
    "symbol": "BTCUSDT",
    "timeframe": "4h", 
    "start_date": "2023-01-01",
    "end_date": "2023-12-31",

    Generated Pine Script Example¶

    Actual code generated by the system:

    // @version=5
strategy("MovingAverageCrossover - Optimized", overlay=false)

// Optimized Parameters (from real optimization - 45.2% returns!)
fast_period = input.float(12, 'Fast Period')
slow_period = input.float(26, 'Slow Period') 
signal_threshold = input.float(0.02, 'Signal Threshold')

// Performance Metrics (from optimization)
// Sharpe Ratio: 1.85
// Total Return: 45.2%
// Max Drawdown: 12.5%

// Moving averages
fast_ma = ta.sma(close, int(math.max(5, 12)))
slow_ma = ta.sma(close, int(math.max(10, 26)))

// Trading signals
long_condition = ta.crossover(fast_ma, slow_ma)
short_condition = ta.crossunder(fast_ma, slow_ma)

// Strategy execution
if long_condition
    strategy.entry("Long", strategy.long)
if short_condition  
    strategy.entry("Short", strategy.short)

// Visual indicators
plotshape(long_condition, "Long", shape.triangleup, location.belowbar, color.green)
plotshape(short_condition, "Short", shape.triangledown, location.abovebar, color.red)

    ‚ö†Ô∏è Error Handling¶

    Common Error Codes¶

    Code HTTP Status Description Solution
    MISSING_API_KEY 401 No API key provided Add X-API-Key header
    INVALID_API_KEY 401 API key not recognized Use valid API key
    RATE_LIMIT_EXCEEDED 429 Too many requests Wait and retry
    INVALID_PARAMETER 400 Invalid request parameter Check parameter values
    OPTIMIZATION_FAILED 500 Optimization job failed Check logs, retry
    EXPORT_FAILED 500 Export generation failed Retry with valid results
